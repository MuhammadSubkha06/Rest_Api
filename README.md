# 📚 JSON REST API - Android Studio Kotlin Training Project

REST API sederhana yang dibuat untuk latihan pengembangan aplikasi Android menggunakan **Kotlin** di **Android Studio**.  
Project ini digunakan sebagai backend API untuk simulasi aplikasi mobile seperti login, registrasi, CRUD data, pencarian data, dan integrasi networking menggunakan Retrofit/Volley.

---

# 🚀 Fitur

- REST API berbasis JSON
- Endpoint CRUD
- Login & Register sederhana
- Response format JSON
- Mendukung integrasi Android Kotlin
- Cocok untuk latihan:
  - Retrofit
  - Coroutine
  - RecyclerView
  - MVVM
  - API Parsing
  - Authentication dasar

---

# 🛠️ Teknologi yang Digunakan

## Backend
- JSON REST API

## Android Client
- Kotlin
- Android Studio
- Retrofit
- Gson Converter
- Coroutine
